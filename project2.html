Introduction :

There are many different DC motor types in the market and all with it good and bad attributes. One such bad attribute is the lag of efficiency. In order to overcome this problem a controller is introduced to the system. There are also many types of controllers used in the industry, one such controller is PID controller. PID controller or proportional–integral–derivative controller is a generic control loop feedback mechanism widely used in industrial control systems. A PID controller attempts to correct the error between a measured process variable and a desired set point by calculating and then outputting a corrective action that can adjust the process accordingly. So by integrating the PID controller to the DC motor we will be able to correct the error made by the DC motor and control the position of the motor to the desired speed or point.
