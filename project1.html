Smart Container

Abstract :

The main objective of this project is to develop a Smart Container Level monitoring system using an UltraSonic sensor [HCSR04] and NodeMcu ESP8266 board with Wi-Fi being remotely controlled by any smart phone for tracking the level of container stored to assist in inventory management without human interaction. The intelligent container is an application of Internet of Things (IoT) in various fields ranginf from kitchen automation to industrial container management. This project incorporates an ultrasonic sensor that uses pulse echo method to monitor and sense the level of commodities kept within storage containers. Pulse echo method determines the difference between an empty container and a particular level of completion based on frequency values taken as threshold values. A NodeMcu ESP8266 microcontroller monitors the level of commodity with the help of sensors and assists in tracking the quantity of commodity.The NodeMcu ESP8266 microcontroller sends the data to AWS IOT Core via MQTT protocol and AWS run analytics on the received data and notifies the user using AWS SNS service. The smart and intelligent container provides personal assistance in monitoring the level of container without human interaction from any place at any time.
